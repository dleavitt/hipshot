Usage:

	hipshot [path-to-config-file]

Config file should look like this:

	{
	  "hipchat": {
	    "token":"f123",
	    "room":"123"
	  },
	  "beanstalk": {
	    "domain":"abc",
	    "username":"abc",
	    "password":"abc"
	  }
	}
